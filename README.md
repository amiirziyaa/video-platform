# Video Platform API

یک وب‌اپلیکیشن اشتراک ویدیو با استفاده از Django و Django REST Framework.

## امکانات کلیدی

- ثبت‌نام و احراز هویت کاربران با JWT
- مدیریت کامل ویدیوها شامل دسته‌بندی، انتشار و مشاهده اطلاعات آماری لحظه‌ای
- تعریف پلن‌های اشتراک، خرید، تمدید و لغو اشتراک با شبیه‌سازی درگاه بانکی
- مدیریت پرداخت‌ها، تاریخچه تماشا، امتیازدهی و نظرات کاربران
- کنترل دسترسی محتوا براساس سطح اشتراک کاربران و قابلیت بوکمارک

## شروع سریع

1. ریپو را کلون کرده و وارد پوشه پروژه شوید.
2. دستورات زیر را اجرا کنید تا محیط مجازی ساخته شده و وابستگی‌ها نصب شوند.

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

3. با اجرای مایگریشن‌ها پایگاه‌داده را آماده کنید و یک اکانت مدیریت بسازید.

```bash
python manage.py migrate
python manage.py createsuperuser
```

4. برای مشاهده نمونه داده و شروع سریع‌تر، دستور زیر را اجرا کنید تا کاربر، پلن و چند ویدیو ایجاد شود:

```bash
python manage.py seed_demo
```

اکنون می‌توانید با نام کاربری `demo` و رمز عبور `demo1234` وارد شوید و API را بررسی کنید.

## اجرا

پس از انجام مراحل بالا، سرور توسعه را با دستور زیر اجرا کرده و از طریق آدرس `http://127.0.0.1:8000/` یا مسیرهای API ادامه‌ی سند به سیستم دسترسی داشته باشید.

```bash
python manage.py runserver
```

## تست

```bash
python manage.py test
```

## مستندات API

پس از اجرای سرور می‌توانید از آدرس `/api/` برای کار با API استفاده کنید. برای دریافت توکن JWT از مسیرهای زیر کمک بگیرید:

- `POST /api/token/` برای دریافت توکن دسترسی و رفرش
- `POST /api/token/refresh/` برای تمدید توکن دسترسی

سایر مسیرهای مهم:

- `POST /api/users/` ثبت نام کاربر جدید
- `GET /api/users/me/` دریافت اطلاعات کاربری
- `GET /api/plans/` مشاهده پلن‌های فعال
- `POST /api/subscriptions/` خرید اشتراک جدید
- `POST /api/subscriptions/{id}/renew/` تمدید اشتراک
- `POST /api/subscriptions/{id}/cancel/` لغو اشتراک
- `GET /api/videos/{slug}/live_status/` مشاهده وضعیت لحظه‌ای ویدیو
- `POST /api/videos/{slug}/watch/` ثبت تماشا و امتیازدهی
- `POST /api/videos/{slug}/comment/` ثبت نظر

این پروژه برای الهام‌گیری از سرویس فیلیمو طراحی شده و ساختار آن به‌گونه‌ای است که بتواند برای توسعه‌های بعدی نیز گسترش پیدا کند.

## رابط کاربری نمونه

علاوه‌بر API، یک رابط کاربری ساده با استفاده از templateهای Django نیز فراهم شده است تا بتوانید جریان کار را سریع‌تر مشاهده کنید:

- صفحه اصلی و لیست ویدیوها در `/` و `/videos/`
- نمایش جزئیات هر ویدیو در مسیر `/videos/<slug>/`
- صفحه پلن‌های اشتراک در `/plans/`
- ثبت‌نام و ورود کاربران در `/accounts/register/` و `/accounts/login/`
- داشبورد شخصی کاربران وارد شده در `/dashboard/`

این صفحات صرفاً برای پیش‌نمایش امکانات فراهم شده‌اند و همچنان API منبع اصلی ارتباط با سرویس به شمار می‌آید.
